# Luneva – PCOS Wellness Mobile Application

Luneva is a cross-platform wellness mobile application designed to support women managing PCOS (Polycystic Ovary Syndrome). Built using Flutter and Firebase, the app combines AI-powered assistance, fitness guidance, diet planning, stress management, and community features into one seamless experience.

Luneva focuses on empowering users through personalized wellness tools, supportive interactions, and an engaging user experience.

---

## Features

- Personalized onboarding experience
- AI-powered chatbot for PCOS guidance
- Diet planning and hydration tracking
- Fitness and workout guidance
- Stress management support
- Real-time chat functionality
- Community and peer support
- User dashboard for wellness tracking
- Smooth animations and modern UI design

## Project Objectives

### Personalized Onboarding

Track meals, hydration, and healthy lifestyle goals.

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/luneva.git
```

### Move Into Project Folder

```bash
cd luneva
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```
